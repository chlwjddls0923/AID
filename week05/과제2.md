5주차. 과제
과제명: 파라미터 및 launch 파일을 활용한 메시지 주기 제어 퍼블리셔 노드 구현



📌 과제 내용

1차 과제를 확장하여, 메시지 출력 주기를 파라미터로 제어할 수 있도록 구현하세요.

파라미터 이름은 interval로 하며, 기본값은 1.0초로 설정합니다.

실행 시 다음과 같이 파라미터를 변경할 수 있어야 합니다:

ros2 run <your_package> hello_param_node --ros-args -p interval:=0.5

또한, launch 파일을 사용하여 노드를 실행하고 파라미터를 전달해보세요.



📂 제출 구성

보고서 1부 (A4 3쪽 이내- 코드 제외)

파이썬 코드 (hello_param_node.py) 전체를 보고서 안에 포함

launch 파일 (hello_launch.py) 전체 포함 및 설명

파라미터 선언 및 적용 방식 설명 (declare_parameter, get_parameter)

실행 방법 2가지:

- ros2 run 명령어로 파라미터 변경

- launch 파일로 실행 시 파라미터 전달 방법

실행 스크린샷 첨부 (예: 0.5초 주기와 2초 주기의 결과 비교)

실행 결과 비교 및 느낀 점
