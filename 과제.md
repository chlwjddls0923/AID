과제명: “Hello ROS2”를 1초마다 출력하는 퍼블리셔 노드 구현



📌 과제 내용

ROS2의 rclpy를 사용하여 노드를 작성하세요.

노드는 1초마다 "Hello ROS2"라는 메시지를 출력해야 합니다.

출력은 get_logger().info()를 이용하며, 타이머는 create_timer()로 설정합니다.



📂 제출 구성

보고서 1부 (A4 3쪽 이내- 코드 제외)

파이썬 코드 (hello_node.py) 전체를 보고서에 포함

```python
import rclpy
from rclpy.node import Node

class HelloNode(Node):
    def __init__(self):
        super().__init__('hello_node')  # 노드 이름: hello_node
        self.timer = self.create_timer(1.0, self.timer_callback)  # 1초 타이머

    def timer_callback(self):
        self.get_logger().info('Hello ROS2')  # 로그 출력

def main(args=None):
    rclpy.init(args=args)
    node = HelloNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

코드 설명: 노드 이름, 구조, 타이머 사용 방식

- 노드 이름: hello_node
- 구조: HelloNode 클래스는 Node를 상속받아 ROS2 노드를 구현합니다. __init__ 메서드에서 노드를 초기화하고 타이머를 생성하며, timer_callback 메서드에서 로그를 출력합니다.
- 타이머 사용 방식: create_timer(1.0, self.timer_callback)를 사용하여 1초마다 timer_callback 함수를 호출합니다.
